Welcome to Coding for Kids Lesson 1!

Today we will be learning about the most basic HTML tag that is the cornerstone of every HTML script.

The first HTML tag we will use is "html". These tags are enclosed in < > signs in the code, but I won't use it here due to the fact the computer might think it is real code.
In this lesson and the following ones, I will use quotation marks instead. This should enclose the entire program, so you should write all the code inside these two tags. The beginning and end "html" tags are different, as the beginning one is just "html", while the end one is written with a slash in the front "/html". Be sure not to confuse these up!

For reference, it should look like this.

Beginning: "html"

End: "/html"
