In this lesson, we will be learning about adding fonts

In order to add a font, follow the steps given below:

1. Go to Google Fonts

2. Search the font you would like to use and click on the button that says "Embed"

3. Copy the code from the embed button and paste it in the head tag before the CSS

This way you can change the font on any given element in your CSS

Example: To add the font MedeivalSharp use this code: "link href="https://fonts.googleapis.com/css2?family=MedievalSharp&display=swap" rel="stylesheet""

Please practice the HTML tags we have taught to ensure that you understand the concept very well.
